HTML

<!DOCTYPE html> is used to declare a Html 5 file.
All the content that has to do with html and the layout itself belongs into the <html></html> tag.
The sections on the top of the page like <title> and the metainformation <meta charset="utf-8"> belong into the header.
Another crucial thing that is done in the header is linking to the CSS file which will be requiered later on.

            
              <head>
                <meta charset="utf-8">
                <title>Hello World!</title>
              </head>

The <body> contains the actual content of the Page. A short example of a webpage: 

            
              <!DOCTYPE html>
              <html>
                <head>
                  <meta charset="utf-8">
                  <title>Hello World!</title>
                </head>
                <body>
                  <article class="Hello_World">
                    <section>
                      <h1>Hello World</h1>
                      <p><em>Hello</em> <strong>World!</strong></p>
                      <pre style="white-space: pre-line">
                        <code>
                          Hello
                          World!
                        </code>
                      </pre>

                    </section>
                  </article>
                </body>
              </html>

CSS

We already have created the basic layout.
Now its time for styling the webpage using the external Stylesheet. In our case it's called stylesheet.css.

HTML

            
              <!DOCTYPE html>
              <html>
                <head>
                  <meta charset="utf-8">
                  <title>Hello World!</title>
                  <link rel="stylesheet" href="stylesheet.css">
                </head>
                <body>
                  <article class="Hello_World">
                    <section>
                      <h1>Hello World</h1>
                      <p><em>Hello</em> <strong>World!</strong></p>
                      <pre>
                        <code>
                          Hello
                          World!
                        </code>
                      </pre>
                    </section>
                  </article>
                </body>
              </html>

CSS

            
              body {
                font-family: monospace;
              }
              h1 {
                border-left : 5px solid red;
                text-indent: 5px;
                background-color: lightgray;
              }
              article p {
                text-align: center;
                text-shadow: 2px 2px lightgrey;
              }
              article code {
                  color: red;
              }
